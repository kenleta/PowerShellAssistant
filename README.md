# 🚀 Universal PowerShell Smart Assistant (v2.5)

A lightweight, intelligent script to supercharge your PowerShell experience. This assistant automates module installation for missing commands and fixes common "path with spaces" errors in real-time.

---

## 🌟 Features / 核心功能

### 1. 📂 CD Smart-Fix (Path Auto-Quoting)
Stop worrying about quotes! If you copy a path from Explorer and it contains spaces, just type \cd \ and paste it. The assistant will catch the error and fix it automatically.
* **Input:** \cd C:\Program Files\Common Files\
* **Result:** Successfully jumps to the directory without manual quotes.

### 2. 📦 Auto-Module Installer & Retry
When you type a command that isn't installed (e.g., \Connect-AzAccount\), the assistant will:
1. Search the **PowerShell Gallery**.
2. Identify the required module.
3. Ask if you want to install it.
4. **Auto-Retry** your original command after installation.

---

## 🚀 One-Liner Installation / 一键部署

Simply copy and paste this command into your PowerShell terminal to deploy the assistant globally on any networked PC:

```powershell
iex (irm bit.ly/4bApAGU)
```

## 🏮 中文快速说明说明

这是一个为您解决 PowerShell 日常痛点的智能辅助脚本：

- **空格路径修正**: 直接执行 \cd\ 带空格的路径，脚本会自动识别并修正，无需手动加引号。
- **缺失命令自动安装**: 输入未安装命令时，自动联网搜索匹配模块，询问安装并在安装后**自动重试**。
- **多版本环境适配**: 完美支持 PS 5.1 和 PS 7.x，自动处理 OneDrive 同步路径。
- **防中断设计**: 按下 \Ctrl+C\ 退出提示后，不会影响后续操作。
